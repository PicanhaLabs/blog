# PicanhaLabs [![Build Status](https://travis-ci.org/PicanhaLabs/blog.svg?branch=master)](https://travis-ci.org/PicanhaLabs/blog)
## Install PicanhaJS

```
npm install -g picanhajs
```

## Clone

```
git clone https://github.com/PicanhaLabs/blog.git picanhablog
```


## Generate

```
cd picanhablog
picanha grill
```

OR


```
cd picanhablog
picanha grill dev
```
